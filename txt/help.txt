<table><tr><td align="left">
<a name="input"><H3>Input Fields</H3></a>
</td></tr></table>


<table>
<tr><td align ="left"> 


<ul>
<li><b><font color=blue>Email address:</font></b> the link with the
results of your request is sent to this address. Using this link you
can view the scoring results.

<li><b><font color=blue>Protein coordinate:</font></b> upload file in <a
href="http://www.rcsb.org/pdb/static.do?p=file_formats/pdb/index.html">PDB format</a>. 

<li><b><font color=blue>Ligand coordinate:</font></b> Upload file in <a
href="http://tripos.com/index.php?family=modules,SimplePage,Mol2_File_Format2009">Mol2 format</a>. 

<li><b><font color=blue>Score type:</font></b> Pose for PoseScore, selecting
near-native geometry from conformations of the same ligand (complex structure
prediction) <b>or</b> Rank for RankScore, selecting ligands from non-binders
(virtual screening)


</ul>
</table>


<table><tr><td align="left">
<a name="input"><H3>Output Field</H3></a>
</td></tr></table>

<table>
<tr><td align ="left">

The output of Pose & Rank is a table (score.list). Each table line contains the name of each uploaded small molecule and the corresponding PoseScore or RankScore.  
<p><b><font color="#000099">Table Format:</font></b><p>
<ul>
<li><b><font color=blue>Ligand name:</font></b> the name of the small molecule
scored that should be in the next line after each "@&#60TRIPOS>MOLECULE" in the
uploaded mol2 file.

<li><b><font color=blue>Score:</font></b> PoseScore or RankScore [1].

</ul>

</table>
